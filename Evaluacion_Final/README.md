# Evaluación Final — Administrador de Usuarios

## Descripción
Aplicación web que consume la API REST de JSONPlaceholder para administrar información de usuarios. Implementa una clase JavaScript (`AdminUsuarios`) que obtiene los datos mediante `XMLHttpRequest` y expone métodos para consultar y mostrar la información en pantalla.

## Objetivos
- Consumir una API REST usando `XMLHttpRequest`
- Implementar el paradigma de clases en JavaScript
- Manipular y filtrar datos con métodos de array (`.map()`, `.find()`, `.sort()`)
- Mostrar resultados dinámicamente en el DOM

## Funcionalidades
1. **Listar nombres** — Muestra los nombres de todos los usuarios
2. **Info básica** — Busca un usuario por nombre y muestra su username y correo
3. **Dirección** — Busca un usuario por nombre y muestra su dirección completa
4. **Info avanzada** — Busca un usuario por nombre y muestra teléfono, sitio web y compañía
5. **Compañías** — Lista todas las compañías con su frase clave (catchphrase)
6. **Ordenar A-Z** — Lista los nombres ordenados alfabéticamente

## Tecnologías
- HTML5
- CSS3
- JavaScript (ES6 — clases, arrow functions)
- API: https://jsonplaceholder.typicode.com/users

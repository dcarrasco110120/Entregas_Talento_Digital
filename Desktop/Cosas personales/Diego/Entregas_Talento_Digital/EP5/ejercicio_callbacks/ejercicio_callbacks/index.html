Funciones Callback en JavaScript

Ejercicio Práctico — Talento Digital

Función 1: validar_numero(callback)

Solicita un número mediante prompt() y valida si es numérico con isNaN(). Llama al callback con el mensaje de éxito o error.

Función 2: calcular_y_avisar_despues(numero, callback)

Calcula la sumatoria de números impares entre 1 y numero. Espera 5 segundos con setTimeout() y luego ejecuta el callback.


Función 3: calcular_y_avisar_dependiendo(numero, callback, callback_error)

Calcula las sumatorias sucesivas desde 1 hasta numero y acumula el total. Si el resultado es menor a 1000 ejecuta el callback; si es 1000 o más, ejecuta el callback_error.

Prueba con número ≤ 17 para resultado < 1000, o ≥ 18 para resultado ≥ 1000.
